# Proyecto Final Backend

Proyecto de Backend para la presentación final en el curso de Backend de la plataforma CoderHouse.
El proyecto pretende *emular* un ecomerce, centrándose en las funciones propias del Backend como el un sistema de log-in y el manejo interno del stock de productos y distintos roles de usuarios con la capacidad de añadir, eliminar productos y manejo de los datos en general.
Incluye una función de chat en vivo usando WebSockets.
Incluye una implementación simple de frontend utilizando Handlebars.

## Tecnologias

Tomando como base Express:

- Librerias como Passport, Handlebars y Nodemailer.
- Manejo de los datos con MongoDB y librerias relacionadas.
